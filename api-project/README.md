# 分布式系统 - 作业1

本项目实现了一个基于AWS无服务器架构的REST API，使用CDK框架配置和部署所需资源。

## 项目上下文

本API用于管理电影信息，提供以下功能：
- 添加新电影
- 获取电影列表
- 按类别筛选电影
- 更新电影信息
- 获取电影描述的多语言翻译

## 技术堆栈

- AWS CDK (TypeScript)
- AWS Lambda
- API Gateway
- DynamoDB
- AWS Translate

## 开发阶段

### 阶段1：初始项目设置与基础架构
- 初始化CDK项目
- 设计和实现DynamoDB表
- 创建基本的API结构

### 阶段2：基本功能实现
- 实现POST请求添加电影
- 实现基本GET请求获取电影列表

### 阶段3：功能扩展
- 实现查询参数筛选的GET请求
- 实现PUT请求更新电影信息
- 实现翻译功能

### 阶段4：安全与授权
- 添加API密钥授权
- 保护POST和PUT端点

### 阶段5：性能优化
- 实现翻译结果缓存
- 优化DynamoDB查询
